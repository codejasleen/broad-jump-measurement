# BroadJump

Simple Python project with `model.py`.

## Description

This repository contains the main script `model.py`.

## Model

The project includes a simple model that calculates the distance of a straight broad jump.

**Note:** This model only accepts a single input: the athlete's height. `model.py` expects the height (in meters) and uses that value to estimate the broad jump distance, then prints or returns the result in meters.

### Inputs

- **height**: Athlete height in centimeters (float). Example: `165`.

### Output

- Estimated broad jump distance in meters (float).

## Requirements

- Python 3.10+

## Setup (Windows)

1. Create and activate a virtual environment:

```bash
python -m venv venv
venv\\Scripts\\activate
```

## Usage

Run the main script:

```bash
py -3.10 model.py
# or
python model.py
```
